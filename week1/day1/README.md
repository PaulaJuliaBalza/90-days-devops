# Día 1: DevOps y la importancia de Linux

## Comandos básicos
```
# Visualizar información del SO y del kernel
uname -a
Linux ASC 5.15.0-139-generic #149~20.04.1-Ubuntu SMP Wed Apr 16 08:29:56 UTC 2025 x86_64 x86_64 x86_64 GNU/Linux

# Mostrar el nombre del usuario actual con el que estoy logueada
whoami
administrator

# Listar archivos y carpetas del directorio actual
ls -lah
-l formato detallado
-a incluir archivos ocultos (quie empiezan con .)
-h mostrar los tamaños en formato legible (KB, MB, etc)

# Muestra hace cuanto está encendido el sistema
uptime

# Visualizar los procesos que consumen más recursos
top

# Visualizar la memoria disponible
free -h
```
